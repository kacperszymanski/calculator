Usage
To use the calculator, simply click on the number buttons and operation buttons to input numbers and perform operations. Alternatively, you can also use your keyboard to input numbers and operations.

Keyboard Shortcuts
The following keyboard shortcuts are available:

Number keys (0-9): Input numbers
Decimal point (.) or comma (,): Input decimal numbers
Addition key (+): Add numbers
Subtraction key (-): Subtract numbers
Multiplication key (*) or X or x: Multiply numbers
Division key (/) or : or รท: Divide numbers
Equal key (=) or Enter: Calculate result
Clear key (C) or Escape: Clear input
Delete key or Backspace: Delete the last digit of the input
Error Handling
The calculator will display an error message if the user attempts to divide by zero.
